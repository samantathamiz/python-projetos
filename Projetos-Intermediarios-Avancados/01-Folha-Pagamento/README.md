# Sistema de Folha de Pagamento em Python
Este projeto é um sistema simples de folha de pagamento desenvolvido em Python, com interface gráfica utilizando Tkinter e banco de dados SQLite.

O sistema permite cadastrar cargos, cadastrar funcionários, calcular salários e gerar relatórios em PDF.


## Funcionalidades
- Cadastro de cargos;
- Cadastro de funcionários;
- Cálculo da folha de pagamento;
- Geração de relatório em PDF;
- Armazenamento em banco de dados SQLite.


## Tecnologias utilizadas
- Python;
- Tkinter;
- SQLite;
- FPDF.


## Objetivo
Projeto desenvolvido para prática de programação em Python, organização de código em múltiplos arquivos e uso de banco de dados.

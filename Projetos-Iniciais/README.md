# Projetos Iniciais em Python

Esta pasta contém exercícios e pequenos projetos desenvolvidos em Python, com o objetivo de praticar lógica de programação e aprendizado da linguagem.

Os projetos incluem ferramentas e simuladores simples, como:

- Calculadora;
- Conversor de unidades;
- Simulador de dados;
- Analisador de texto;
- Entre outros exercícios de aprendizado.

Todos os projetos foram desenvolvidos para estudo e prática de conceitos básicos de Python, e cada exercício está em sua própria pasta ou arquivo.

## Tecnologias utilizadas

- Python 3;
- Bibliotecas padrão do Python.
